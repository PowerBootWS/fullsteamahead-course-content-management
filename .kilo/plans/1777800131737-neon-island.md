# Plan: Disable the Kilo Provider

## Goal

Prevent the built-in `kilo` AI provider from loading in this project by adding it to `disabled_providers` in the project's Kilo config.

## Background

Per the [Kilo AI Providers docs](https://kilo.ai/docs/ai-providers#disabling-built-in-providers), providers can be disabled by listing their IDs in `disabled_providers` inside `kilo.json` / `kilo.jsonc`.

The provider ID for the Kilo built-in provider is `"kilo"`.

## Current State

`.kilo/kilo.jsonc` currently contains only the schema reference:

```jsonc
{
  "$schema": "https://app.kilo.ai/config.json"
}
```

## Change

Edit `.kilo/kilo.jsonc` to add `"disabled_providers": ["kilo"]`:

```jsonc
{
  "$schema": "https://app.kilo.ai/config.json",
  "disabled_providers": ["kilo"]
}
```

## File to Edit

- **`.kilo/kilo.jsonc`** — Add `"disabled_providers": ["kilo"]` field.

## Steps

1. Edit `.kilo/kilo.jsonc` — insert `"disabled_providers": ["kilo"]` alongside the existing `$schema` key.
